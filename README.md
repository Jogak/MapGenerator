# MapGenerator

![Diagramme de classe](uml/structure.puml)

## Paramètres

Le fichier *Parameters.js* contient tous les paramètres liés à la génération de la map.

###  TEMPERATURE

Permet de changer le jeu de tuile.

| Valeur |  Jeu  |
|-----|----------|
|  -1 | Glacial  |
|   0 |  Tempéré |
|   1 | Savane   |
|   2 | Tropical |
|   3 | Desert   |
